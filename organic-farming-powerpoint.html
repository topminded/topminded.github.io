2. what is the importance of powerpoint presentation in an organization?​


What is the importance of powerpoint presentation in an organization?​

Powerpoint is a presentation / slideshow application software which allows the user to create slideshows for lectures, presentation, meetings and other purpose. It serves as a digital visual aid. Traditionally, when teachers and presenters would like to discuss something, they would use cut outs, boards and papers with their lectures written on it. Powerpoint transform that idea into a digital slide.  

Powerpoint is important in an organization especially when you are going to meet some outsiders or you would like to present your product to a potential client. Creating slideshows can give your client an impression about your company. Also, it would not make the discussion boring as they could see what you are talking in your discussions thru images and figures you are presenting on the slides. You don't need to discuss everything on the slides, which allows you to discuss more details other than what is written on the slides. Lastly, it can run on any Windows PC, mobile and tablets.

The most distinguish feature of slideshows made in Powerpoint is the "Custom Animation", which could be used to add animation effects on the objects found in the slides. Such objects includes: texts, images, cliparts, tables and others. Aside from Custom Animation, it has slide transition which you could edit and assign an effect of the slide exit and entry.

23. III. Directions: Write the steps on how to add ANIMATIONS in Microsoft PowerPoint Write your answerinside the graphic organizer. (16-20 5pts.)​


Answer:

Add animations and effects:

1. Select the object or text you want to animate.

2. Select Animations and choose an animation.

3. Select Effect Options and choose an effect.

Manage animations and effects:

There are different ways to start animations in your presentation;

On Click - Start an animation when you click a slide.

With Previous - Play an animation at the same time as the previous animation in your sequence.

After Previous - Start an animation immediately after the previous one happens.

Duration - Lengthen or shorten an effect.

Delay - Add time before an effect runs.

Add more effects to an animation:

Select an object or text with an animation.

Select Add Animation and choose one.

Change the order of animations:

1. Select an animation marker.

2. Choose the option you want:

A. Move Earlier - Make an animation appear earlier in the sequence.

B. Move Later - Make an animation occur later in the sequence.

C. Add animation to grouped objects

You can add an animation to grouped objects, text, and more.

D. Press Ctrl and select the objects you want.

E. Select Format > Group > Group to group the objects together.

F. Select Animations and choose an animation.
