Answer:

968cm²

Explanation:

l = 2b

2(l + b) = 132

2 [(2b) + b] = 132

2 [3b] = 132

6b = 132

b = 132/6

b = 22

l = 2b

l = 2[22]

l = 44

area = 22 × 44 = 968cm²
