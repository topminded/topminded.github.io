1. When can we say that an object is in motion?How do we describe the motion of an object?​


Answer:

When can we say that an object is in motion?

A reference point is a place or object used for comparison to determine if something is in motion. An object is in motion if it changes position relative to a reference point. You assume that the reference point is stationary, or not moving.

How do we describe the motion of an object?

You can describe the motion of an object by its position, speed, direction, and acceleration. An object is moving if its position relative to a fixed point is changing. Even things that appear to be at rest move.

Question:

When can we say that an object is in motion?

Answer:

An object is in motion when its distance from another object is changing. Whether an object is moving or not depends on your point of view. For example, a woman riding on a bus is not moving in relation to the seat she is sitting on, but she is moving in relation to the buildings the bus passes.

25. when can we say that an object isin motion​


Answer:

An object is said to be in motion when it changes its position with time with respect to its surroundings.

Explanation:

To say that an object is in motion, first we need to define a frame of reference. This requires establishing a set of physical reference points and deciding which direction will be considered positive, also called coordinate system. This will allows to identify the position of the object at any time. Once this has been defined, we can say that an object is in motion if its position is changing with respect to time.
