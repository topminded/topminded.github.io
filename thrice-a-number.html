7. 4) The sum of twice one number and thrice another number is 18. The sum of thrice the first number and twice the second number is 7. Find the numbers.​


Answer:

x = 6, y = 1

Step-by-step explanation:

Let's call the first number "x" and the second number "y".

From the first given information:

2x + 3y = 18

From the second given information:

3x + 2y = 7

Now we have a system of two equations with two variables. We can use either substitution or elimination method to solve it.

1. Substitution method:

Solve one of the equation for one variable.

3x + 2y = 7

3x = 7 - 2y

x = (7 - 2y) / 3

Substitute this value of x into the other equation and solve for y.

2x + 3y = 18

2((7 - 2y) / 3) + 3y = 18

14 - 4y + 3y = 18

-y = 4

y = -4

Substitute this value of y into the equation for x that you found earlier.

x = (7 - 2y) / 3

x = (7 - 2(-4)) / 3

x = (7 + 8) / 3

x = 15 / 3

x = 5

So, the two numbers are x = 5 and y = -4.

2. Elimination method:

Multiply one equation by a number that will eliminate one of the variables when added to the other equation

3x + 2y = 7

2x + 3y = 18

6x + 6y = 42

Now we have an equation with one variable, we can solve it as:

6x + 6y = 42

6y = 42 - 6x

y = (42 - 6x) / 6

Substitute this value of y into the first equation, and solve for x

2x + 3((42 - 6x) / 6) = 18

2x + (42 - 6x) = 18

2x + 42 - 6x = 18

-4x = -24

x = 6

Substitute this value of x into the equation for y, and find the value of y

y = (42 - 6x) / 6

y = (42 - 6(6)) / 6

y = (42 - 36) / 6

y = 6 / 6

y = 1

So, the two numbers are x = 6 and y = 1.
