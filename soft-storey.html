5. an object is dropped from rest from a 15 storey building 1 storey = 4.25 meters how long will it take from for the object to reach the ground​


Solution:

In this problem, we will use the second kinematic equation of vertical motion. In formula form,

y = (vi)(t) + (1/2)(gt²)

where:

y = vertical distance (usually the height of a building)

vi = initial velocity

t = time

g = acceleration due to gravity

g = 9.81 m/s²

Listing the given values

y = 15(4.25) = 63.75 m

vi = 0 (The object is dropped.)

Solving for t

y = (vi)(t) + (1/2)(gt²)

63.75 = (0)(t) + (1/2)(9.81)(t²)

63.75 = (4.905)(t²)

t² = 63.75 / 4.905

t = √(63.75 / 4.905)

t = 3.61 s

Therefore, the time taken by the object to reach the ground is 3.61 s.

29. Is mezzanine applicable for two storey house?


Answer

A Mezzanine can be built in any structure as long as it doesn't violate the National Building Code of the Philippines.

Explanation

According to the National Building Code of the Philippines (NBCP) Republic Act (RA) 6541, Mezzanine floors shall have a clear ceiling height not less than 1.80 meters above and below it. Meaning, a floor should have a ceiling height of at least 3.70 to 4 meters to build a Mezzanine following Philippine standards.

Additional Information

A Mezzanine is an open space platform between the floor and the ceiling. It is usually seen in industrial structures, warehouses, malls, and modern townhouses. This is because a Mezzanine can help add more space on a floor level.
