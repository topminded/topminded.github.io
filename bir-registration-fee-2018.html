16. The cost of membership in a fitness camp includes a monthly charge and a one-time registration fee. Find the monthly charge and the registration fee. 3 months-Php 8 5006 months-Php 14 500​


Let the monthly charge be represented by x and the registration fee by y.

From the given information, we can set up two equations:

For 3 months: 3x + y = 8500

For 6 months: 6x + y = 14500

We can solve for x and y using elimination or substitution.

Using elimination:

Multiplying the first equation by 2, we get:

6x + 2y = 17000

Subtracting the second equation from this, we get:

(6x + 2y) - (6x + y) = 17000 - 14500

Simplifying, we get:

y = 2500

Substituting this value of y in one of the equations, we can solve for x:

3x + 2500 = 8500

3x = 6000

x = 2000

Therefore, the monthly charge is Php 2,000 and the registration fee is Php 2,500.
