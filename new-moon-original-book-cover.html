8. 2over 3 x - 1over 6=1over 2 x + 5over 6


Answer:

To solve this equation, we need to simplify both sides and isolate the variable "x" on one side.

First, we can simplify the left-hand side by finding a common denominator for the two fractions:

2/3 x - 1/6 = (4/6)x - 1/6 = (2/3)x - 1/6

Similarly, we can simplify the right-hand side by finding a common denominator:

1/2 x + 5/6 = (3/6)x + 5/6 = (1/2)x + 5/6

Now we can rewrite the original equation as:

(2/3)x - 1/6 = (1/2)x + 5/6

To isolate the variable "x" on one side, we can start by subtracting (1/2)x from both sides:

(2/3)x - (1/2)x - 1/6 = 5/6

Next, we need to find a common denominator for (2/3) and (1/2):

(4/6)x - (3/6)x - 1/6 = 5/6

Simplifying the left-hand side gives:

(1/6)x - 1/6 = 5/6

Adding 1/6 to both sides gives:

(1/6)x = 1

Finally, we can solve for x by multiplying both sides by 6:

x = 6

Therefore, the solution to the equation is x = 6.
