[tex]{\sf{12a^2-44(ab)-45b^2}}[/tex]

[tex]{\sf{12a^2+(10-54)(ab)-45b^2}}[/tex]

3. Apple distributive property and remove unnecessary parenthesis.

[tex]{\sf{12a^2+10ab-54ab+45b^2}}[/tex]

4. Group the first two terms and the last two terms.

[tex]{\sf{(12a^2+10ab)-(54ab+45b^2)}}[/tex]

5. Factor out the greatest common factor (GCF) from each group.

[tex]{\sf{2a(6a+5b)-9b(6a+5b)}}[/tex]

19. A. Simplify the following expressions. 1. b5b³ 2. 1 4 S 3. (-2)³​


Answer:

1. b^5 * b^3 can be simplified by using the rule of exponents that states when multiplying like bases, you add the exponents. Therefore, b^5 * b^3 = b^(5+3) = b^8.

2. 1/4S can be simplified by using the commutative property of multiplication which states that the order of factors does not change the result. Therefore, we can write 1/4S as S/4.
