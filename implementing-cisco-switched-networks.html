4. do all networks need a switch? why? ​


Answer:

Not all networks need a switch. In a small network with only a few devices, a switch may not be necessary because the devices can be connected directly to each other using cables. However, in larger networks with many devices, a switch is necessary because it allows the devices to communicate with each other more efficiently.

Switches are used in networks to create a central connection point for all of the devices on the network. This allows the devices to communicate with each other and access the internet without having to be connected directly to each other. This can greatly improve the performance of the network because it reduces the amount of traffic on the network and allows the devices to communicate more quickly and efficiently.

Additionally, switches can provide advanced features such as network segmentation, which allows network administrators to divide a network into smaller sub-networks for improved security and performance. This can be particularly useful in large networks where different departments or groups of users may need to be isolated from each other for security reasons.

In summary, while not all networks need a switch, they are an essential component of most modern networks because they improve performance and provide advanced features such as network segmentation.

18. How do switch buffers affect network performance?


Answer:

They provide error checking on the data received.

They store frames received, thus preventing premature frame discarding when network congestion occurs.

They provide extra memory for a particular port if autonegotiation of speed or duplex fails.

They hold data temporarily when a collision occurs until normal data transmission resumes.
