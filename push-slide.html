Types of Fault

There are several types of faults or faults which can be categorized based on the type of movement, as follows:

1. Normal Fault

The fault is said to be a normal fault if the fault allows one block, for example the foot wall has a layer that moves in the same direction but rises relatively to the other block, namely the hanging wall. A feature that is very easy to find in this type of normal fault is its slope which is almost 90 degrees.

2. Reserve Fault

The opposite of normal fault, for this type of reserve fault is a fault that occurs with the direction of the foot wall which is relatively down compared to the hanging wall. An easy feature to find in this type of fault is its relatively small slope, which is only about 45 degrees or half of the normal fault slope.

3. Strike Fault

For this type of strike fault, it is a fault that has a relatively horizontal direction either to the left or to the right. The direction of this fault does not all move in a horizontal direction but can also move in a vertical direction. In this type of fault that moves to the left is called dextral while for faults that moves to the right is called sinistral.
