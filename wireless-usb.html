10. Differentiate/compare the following components below. (5 points each) 1. DDR and SDR 2. HDR and SSD 3. USB flash drive and External Hard Drive 4. LAN and wireless connection 5. AMD and Intel​


Answer:

1. DDR and SDR: DDR (Double Data Rate) is a type of memory technology used in computer systems for high-speed data transfer. It operates at a higher frequency compared to SDR (Single Data Rate) and allows for faster processing times. SDR, on the other hand, is an older technology that operates at a slower speed.

2. HDR and SSD: HDR (High Dynamic Range) refers to a display technology that can display a wider range of colors and brightness compared to traditional displays. It provides a more vivid and lifelike image. An SSD (Solid State Drive) is a type of data storage device that uses flash memory to store data. It operates faster and more efficiently compared to traditional hard drives.

3. USB flash drive and External Hard Drive: A USB flash drive is a small, portable data storage device that uses flash memory and connects to a computer via USB. An external hard drive is a larger, standalone data storage device that also uses spinning disks to store data and connects to a computer via USB or other interface.

4. LAN and wireless connection: A LAN (Local Area Network) is a computer network that connects devices within a limited geographic area, such as a home or office. A wireless connection, on the other hand, allows devices to connect to a network without the use of physical cables.

5. AMD and Intel: AMD (Advanced Micro Devices) and Intel are two of the largest manufacturers of microprocessors for computer systems. AMD produces processors for both personal computers and servers, while Intel mainly focuses on the personal computer market. Both companies offer different processor models with varying levels of performance and power consumption.
