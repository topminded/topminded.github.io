- Solving a quadratic inequality in Algebra is similar to solving a quadratic equation. The only exception is that, with quadratic equations, you equate the expressions to zero, but with inequalities, you’re interested in knowing what’s on either side of the zero i.e. negatives and positives.

[tex] \large \sf {x}^{2} - 9x + 14 > 0[/tex]

[tex]\sf {x}^{2} - 9x + 14 = 0[/tex]

[tex] \sf(x - 7)(x - 2) = 0[/tex]

[tex] \sf x - 7 = 0 \: \: \: \: ; \: \: \: \: x - 2 = 0[/tex]

[tex] \sf x = 7 \: \: \: \: ; \: \: \: \: x = 2[/tex]

- Write the solution in inequality notation or interval notation. So since the interval notation is in the attachment photo.

- After plotting the points in the number line, used the test point method to get whether the equation is true or not.
