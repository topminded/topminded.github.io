29. The wooly mammoth became extinct approximately 10,000 yrs ago If a human lifetime is approximately 72yrs,how many lifetime ago did the mammoth become extinct a.13.9 human lifetimes b.139 human lifetimes c.720,000 human lifetimes d.72,000 human lifetime


Answer:

b.139 human lifetimes

Explanation:

1 human lifetime ( or HLt)  = 72 yrs

10000 yrs =  X

72 yrs : 1 HLt  = 10000 yrs : X

72 yrs : 1 HLt  = 10000 yrs : X

72 yrs / 1 HLt  = 10000 yrs / X

X( 72 yrs / 1 HLt )  = 10000 yrs

X = 10000 yrs / (72 yrs / 1 HLt)

X = ( 10000 /72 HLt )

X = 138.889 HLt or  ~ 139 HLt )
