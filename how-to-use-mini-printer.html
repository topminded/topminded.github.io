8. you printed your 40-page reaction paper. You observed that printer A in the computer shop finished printing in two minutes. How long will take printer A to print 150 pages? How long will it take printer A to print p Pages? if printer B can print x pages per minute,how long will take to print p pages?The rate of each printer is constant. and how didyou solved the rate of each printer? how did you compute the time of each printer? what will happen if the rate of the printer increases? how do time and number of pages affect the rate of the printer?


For printer A, 40 page = 2 minutes.

So for printer A, 1 page = 1/20 minutes.

So printer A will take p/20 minutes because 1*p page = 1/20 * p minutes.

Set p into 150, you get 150/20 = 15/2 = 7.5 minutes.

If printer B can print x pages per minute, It will take p/x minutes to print p pages because

x (pages) : 1 (minute) =1:1/x for any x

then 1:1/x = p:p/x for any x and any p


Rate here is measured as amount of page) over time, so the rate (of speed) of printer A is 40/2 = 20 pages per minute,
and the rate of printer B is x pages per minute.

so, as the time decreases, the rate increases because it is the denominator.
as the number of pages increases, the rate increases too because it is the numerator.
