6. 1. Calculate the work done by each crane if the load is 700 kg and the height is 60 m. 2. For crane 1, calculate the power p with time t = 10 minutes. 3. For crane 2, calculate the power P with time t = 5 minutes. 4. Compare the power P for the two cranes.​


Answer:

1. Given

   m = 700kg

    h = 60 m

    g =

   Work = Force (F) x distance (d)

            = Mass (m) x acceleration (g) x distance (d)

            = 700 kg x 9.8 m/[tex]s^{2}[/tex] x 60m

            = 411,600 Nm

            = 411, 600 J

Explanation:

Work is done when force is done on an object and results in the movement of the object at a specific distance. To solve for work, we multiply force with distance. Force, on the other hand, is the mass multiplied by acceleration. The acceleration used here is the acceleration due to gravity since it refers to an object at a specific height. The value of acceleration due to gravity is 9.8 m/[tex]s^{2}[/tex].  Substituting the values to the formula gives us the amount of work done which is equal to 411,600J.

2.  Given

    W = 411,600 J

     t = 10 minutes = 600 seconds

  P = W/t

      = 411,600 J/600s

     = 686 J/s or 686 Watts

3. W = 411,600 J

     t = 5 minutes = 300 seconds

  P = W/t

      = 411,600 J/300s

     = 1372 J/s or 1372 Watts

4. The power of crane 2 is greater than crane 1. It means that crane 2 was able to do the same amount of work as crane 1 but at a shorter period of time.

Explanation:

Power is the rate of work done or the rate at which energy is used. Power is directly proportional to the amount of work but inversely proportional to the time. It means that if the time is equal, the more work being done, the greater power. However, if the work is equal, the shorter the time, the more power the object has. One example to illustrate this is if we compare walking up and running up the same stairs. More power is evident in running up the stairs since one is able to do work in a short period of time.

In the question, crane 2 has more power than crane 1 because crane 2 is able to work in five minutes only while crane 1 is in 10 minutes. As shown in the computation above, the formula for power is equal to work divided by time (P = W/t). The time is converted to seconds in order to arrive at J/s which is equal to watts (W), the SI unit of power.

7. a crane lifts a 1,800 kg load of scrap metals to a distance of 15m in 20 s. How much power is used in delivering the load? with solution please​


[tex]\tt{\huge{\blue{Explanation:}}}[/tex]

The power of an object doing a work W in a given time t is given by

[tex]\boxed{P = \frac{W}{t}}[/tex]

where:

P = power

W = work

t = time

[tex]\tt{\huge{\red{Solution:}}}[/tex]

Solving for the work

W = mgh

W = (1,800 kg)(9.8 m/s²)(15 m)

W = 264,600 J

Solving for the power

[tex]P = \dfrac{W}{t}[/tex]

[tex]P = \dfrac{\text{264,600 J}}{\text{20 s}}[/tex]

[tex]\boxed{P = \text{13,230 W}}[/tex]

Therefore, the power used in delivering the load is 13,230 W.
