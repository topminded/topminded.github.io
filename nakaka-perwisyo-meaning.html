11. sino po nakaka alam ng ibig sabihin ng "Matematical na lohika"? (do any of you know what "Matematical na lohika" means?)​


Answer:

Ang lohikal na estilo ng pag-aaral ng matematika ay isa sa walong uri ng mga estilo ng pag-aaral, o mga katalinuhan, na tinukoy sa teorya ng pag-unlad na psychologist na si Howard Gardner ng Maramihang Mga Intelligence. Ang lohikal na estilo ng pag-aaral ng matematika ay tumutukoy sa iyong kakayahang mangatwiran, lutasin ang mga problema, at matutunan ang paggamit ng mga numero, abstract visual na impormasyon, at pag-aaral ng mga kaugnayan sa sanhi at epekto.


12. paano nakaka apekto ang Climate/weather ng isang lugar at paano nakaka apekto sa Sun​


Ang klima at panahon ay maaaring magdulot ng iba't ibang epekto depende sa lokasyon ng lugar at uri ng klima. Ang mga epekto ng klima ay maaaring mag-include ng pagbabago ng temperatura, pagbaha at pagguho ng lupa, at pagsikat ng araw. Ang mga epekto ng mga pagbabago sa klima ay maaari ring magdulot ng mga problemang may kaugnayan sa kalusugan at ekonomiya ng mga tao, at kalikasan at ekolohiya. Mahalaga na mag-ingat at maghanda sa anumang mga epekto ng klima at panahon upang maprotektahan ang kalusugan at kalikasan.

2. A rectagular plot is 6 meters longer than its width the area of the plot is 16 square meters. Find the length and width of the plot

-Width = 2 meters

Length = 8 meters

Explanation:

Let, width = x

length = x + 6

area = 16 m²

Translate,

Area = Length x Width

16 = x(x + 6)

Use FOIL,

16 = x² + 6x

Rearrange,

x² + 6x - 16 = 0

Factor,

x² + 8x - 2x - 16 = 0

(x - 2) (x + 8) = 0

Find the roots,

x - 2 = 0

x1 = 2

x + 8 = 0

x2 = -8

Disregard the negative root since we are dealing with measurements

Substitute the positive root,

width = 2 meters

length = 2 + 6 = 8 meters
