Different types of values can be in the same array constant — for example, {1,3,4;TRUE,FALSE,TRUE}. Numbers in array constants can be in integer, decimal, or scientific format. Text must be enclosed in double quotation marks — for example, "Tuesday".

Array constants cannot contain cell references, columns or rows of unequal length, formulas, or the special characters $ (dollar sign), parentheses, or % (percent sign).

Enclose them in braces ( { } ).

25. A = (b1+b2)h/2; solve for b1, b2, and h


A = [(b₁ + b₂)h] / 2

Find b₁  and  b₂:

2A = b₁h + b₂h

2A - b₂h = b₁h

b₁ = (2A - b₂h)/h

b₁ = (2A/h) - b₂b₂ = (2A/h) - b₁

Find h:

A = [(b₁ + b₂)h] / 2

2A = (b₁ + b₂)h

h = 2A/(b₁ + b₂)
