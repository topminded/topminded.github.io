14. B. Identification:Identify the following:1. Vector Images2.Sharpening and Softening 3. Photo Manipulation4. Infographics5. Saturation6. Contrast7. Balance8. Marque Tool9. Layers 10. Image Orientation​


Answer:

1. Vector Images: Graphics created using mathematical equations to represent shapes and lines.

2. Sharpening and Softening: Image editing techniques used to enhance or reduce the sharpness of an image.

3. Photo Manipulation: The process of altering an image using various techniques to change its appearance.

4. Infographics: Visual representations of data or information using charts, graphs, and other design elements.

5. Saturation: The intensity of colors in an image.

6. Contrast: The difference between the lightest and darkest parts of an image.

7. Balance: The distribution of elements in a design to create a sense of stability and visual harmony.

8. Marque Tool: A selection tool in image editing software used to select specific areas of an image.

9. Layers: A feature in image editing software that allows users to stack and edit multiple elements of an image separately.

10. Image Orientation: The position of an image, either horizontally (landscape) or vertically (portrait).
