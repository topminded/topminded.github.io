K = [  AlBr₃] / [ Al] [  Br₂]³/²

K² =  [  AlBr₃]² / [  Al ] ² [ Br₂]³

1) 2 AlBr₃ = 2 Al(s) + 3 Br₂(l) =

K₁ =  [  Al ] ² [ Br₂]³ /  [  AlBr₃]²

2 ) 2 Al(s) + 3 Br₂(l) = 2 AlBr₃(s)
