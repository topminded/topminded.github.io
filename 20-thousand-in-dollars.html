4. The following table shows last year's revenue for each Herman Hoagie's location Location: Revenue ( in thousands of dollars)Cedarton 121Rimber 189Linton 147Granton ?If the mean of the data set is 158 thousand dollars, find the revenue for the Granton location.


Mean

Mean shows the average of a number of data. The data collected can be in the form of single data or data with frequency.

Data from the last year's revenue for each Herman Hoagie's location shows a single data. If we count from the mean value, then the revenue for the Granton location is 175

Further explanation

Data is needed in the analysis of a study or experiment  

Data can be collected in the form of values

The data obtained can be expressed in the form of tables, graphs or diagrams  

From the data distribution, it can be seen the frequency of the values

Mean is one way to get the average value from numerical data  

Mean is the average value of some data. The mean value can be determined by dividing the sum of all data by the value of data  

Can be formulated  

[tex]\tt \bar{x}=\dfrac{x_1+x_2+x_3..x_n}{n}[/tex]

[tex]\tt \bar{x}=\dfrac{\sum x_n}{n}[/tex]

Given

Revenue for each Herman Hoagie's location  ( in thousands of dollars)

Cedarton : 121

Rimber : 189

Linton : 147

Mean : 158

Total data = n = 4

Required

the revenue for the Granton location.

Solution

Input the data in the formula, and assume for the Granton location = x

[tex]\tt 158=\dfrac{121+189+147+x}{4}[/tex]

158.4 = 121+189+147+x

632 = 457 + x ⇒combine like terms

x = 632 - 457

x = 175

So the revenue for the Granton location : 175  ( in thousands of dollars)

14. 1. The profit (in thousands of dollars) of a company is given by. P(x) = 5,000 + 1,000 x - 5x2 where x is the amount (in thousands of dollars) the company spends on advertising. a) Find the amount, x, that the company has to spend to maximize its profit by Find the maximum profit P max.​


Answer:

To maximize the profit, we want to find the value of x that maximizes the function P(x). To do this, we need to find the derivative of P(x) and set it equal to 0.

The derivative of P(x) is given by:

P'(x) = 1,000 - 10x

To find the value of x that maximizes the function, we set the derivative equal to 0 and solve for x:

1,000 - 10x = 0

x = 100

Substituting this value back into the original function, we find the maximum profit:

P max = 5,000 + 1,000 * 100 - 5 * 100^2 = 5,000 + 100,000 - 50,000 = $155,000

Therefore, the company needs to spend $100,000 on advertising to maximize its profit, and the maximum profit is $155,000.
