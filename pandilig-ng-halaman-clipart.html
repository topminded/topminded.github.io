13. pandilig Ng halamanpangkahat Ng lupapang tang Ng lupapang bungkal Ng lupapanghalo Ng lupa at pataba​


Ang Lagadera ay ginagamit para sa pandilig ng halaman.

Ang Pala ginagamit pang kahat ng lupa

Katulad ng pangkahat ng lupa syempre madami rin ang uri ng lupa may 6 na uri ng lupa magkakaiba ang tinatanim sakanila depende na lang ano ang iyong itatanim ito ay Clay, Loam, Peaty, Saline, Sandy, at Silty.

Ang Asarol ang ginagamit para sa pangbungkal ng lupa.

Ang Kalaylay ang ginagamit para sa paghalo ng lupa nagagamit rin ito sa pagpapantay ng lupa at paghihiwalay ng bato mula sa lupa. Ang pataba naman marami slang uri katulad ng uri ng lupa pero nagagamit sila anumang lupa ito ito ay ang Organic fertilizer, Manure, Bone meal, Chicken Manure, etc.
