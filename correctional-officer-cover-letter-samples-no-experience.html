To learn the concept of the probability distribution of a discrete random variable.

To learn the concepts of the mean, variance, and standard deviation of a discrete random variable, and how to compute them.

Probability Distributions

Associated to each possible value x of a discrete random variable X is the probability P(x) that X will take the value x in one trial of the experiment.

Definition

The probability distribution of a discrete random variable X is a list of each possible value of X together with the probability that X takes that value in one trial of the experiment.

The probabilities in the probability distribution of a random variable X must satisfy the following two conditions:

Each probability P(x) must be between 0 and 1: 0≤P(x)≤1.

The sum of all the probabilities is 1: ΣP(x)=1.

EXAMPLE 1

A fair coin is tossed twice. Let X be the number of heads that are observed.

Construct the probability distribution of X.

Find the probability that at least one head is observed.

Solution:

The possible values that X can take are 0, 1, and 2. Each of these numbers corresponds to an event in the sample space S={hh,ht,th,tt} of equally likely outcomes for this experiment: X = 0 to {tt}, X = 1 to {ht,th}, and X = 2 to {hh}. The probability of each of these events, hence of the corresponding value of X, can be found simply by counting, to give

xP(x)00.2510.5020.25

This table is the probability distribution of X.

“At least one head” is the event X ≥ 1, which is the union of the mutually exclusive events X = 1 and X = 2. Thus

P(X≥1)=P(1)+P(2)=0.50+0.25=0.75

8. If an experiment consists of throwing a die and then drawing a letter at random from the English alphabet, how many points are in the sample space?


The number of points in the sample space is 156. To get the sample space, you must get the number of possibilities there is with the given situation. There are 6 sides in a dice and 26 letters in the alphabet. The short method is to multiply 6 and 26 to get sample space.

Set S = 6 × 26

Set S = 156

Background

A sample space is the set of the probability of the experiment. The problem asks for the number of possibilities considering the sides of the dice and the letters of the alphabet. Simply: “How many results are there for one side and one letter to come out together?”

Solution (Long Method) Identify the sample of the two different sets. Combine the two sets. Count the number of elements or members in the set.  Answer.  

The sample set of the dice is {1, 2, 3, 4, 5, 6}.  

The sample set of the alphabet is {A, B, C, …, X, Y, Z}

The sample set of the dice and the alphabet is {1A, 1B, 1C, …, 2A, 2B, 2C, …, 5A, 5B, 5C, …, 6A, 6B, 6C}.

The sample set has 156 elements. Therefore, the sample space has 156 points.

18. An experiment consists of rolling a die, and then selecting a letter at random from English alphabet. How many outcomes are there in the sample space for this experiment?


Answer:

Sample space of an experiment or random trial is the set of all possible outcomes or results of the experiment. It is usually denoted by using set notation.

In the problem above;

The sample space of throwing a die is 6. It is {1,2,3,4,5,6}.

The sample space of drawing letters in an English alphabet is 26.

S= sample space of throwing a die x sample space in drawing a letters

 = 6 x 26

 = 156

Therefore, there are 156 points in the sample space.

List them and compute the mean of each sample. Construct the sampling distribution of the sample means. Construct the histogram of the sampling distribution of the sample means.​


Answer:

a) The number of samples possible can be calculated using the formula for combinations. We choose 5 officers out of 6, so the number of samples possible is:

C(6,5) = 6

b) There are six possible samples of size 5:

Sample 1: A, B, C, D, E

Mean = (8 + 12 + 16 + 20 + 24)/5 = 16

Sample 2: A, B, C, D, F

Mean = (8 + 12 + 16 + 20 + 28)/5 = 16.8

Sample 3: A, B, C, E, F

Mean = (8 + 12 + 16 + 24 + 28)/5 = 17.6

Sample 4: A, B, D, E, F

Mean = (8 + 12 + 20 + 24 + 28)/5 = 18.4

Sample 5: A, C, D, E, F

Mean = (8 + 16 + 20 + 24 + 28)/5 = 19.2

Sample 6: B, C, D, E, F

Mean = (12 + 16 + 20 + 24 + 28)/5 = 20

29. 2over 3 x - 1over 6=1over 2 x + 5over 6


Answer:

To solve this equation, we need to simplify both sides and isolate the variable "x" on one side.

First, we can simplify the left-hand side by finding a common denominator for the two fractions:

2/3 x - 1/6 = (4/6)x - 1/6 = (2/3)x - 1/6

Similarly, we can simplify the right-hand side by finding a common denominator:

1/2 x + 5/6 = (3/6)x + 5/6 = (1/2)x + 5/6

Now we can rewrite the original equation as:

(2/3)x - 1/6 = (1/2)x + 5/6

To isolate the variable "x" on one side, we can start by subtracting (1/2)x from both sides:

(2/3)x - (1/2)x - 1/6 = 5/6

Next, we need to find a common denominator for (2/3) and (1/2):

(4/6)x - (3/6)x - 1/6 = 5/6

Simplifying the left-hand side gives:

(1/6)x - 1/6 = 5/6

Adding 1/6 to both sides gives:

(1/6)x = 1

Finally, we can solve for x by multiplying both sides by 6:

x = 6

Therefore, the solution to the equation is x = 6.
