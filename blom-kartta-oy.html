6. O is the midpoint of JY. JO=8x and OY=6x+18. Solve for the values of: x, JO, OY anf JY


Since O is the midpoint of points J and Y, then we can say that JO = OY.


GIVEN:

JO = 8x

OY = 6x + 18


SOLUTION:

JO = OY

Finding the value of x.

8x = 6x + 18

8x - 6x = 18

2x = 18

[tex](2x = 18)\frac{1}{2}[/tex]

[tex]\frac{2x}{2} = \frac{18}{2}[/tex]

x = 9.

Finding the value of the line segment JO.

JO = 8x

JO = 8(9)

JO = 72 units.

Finding the value of the line segment OY.

OY = 6x + 18

OY = 6(9) + 18

OY = 54 + 18

OY = 72 units.

Finding the value of line JY.

JO + OY = JY

72 + 72 = JY

144 units = JY


ANSWER:

Therefore, the value of x is 9, the length of line segment JO is 72 units, length of line segment OY is 72 units, and the length of the line JY is 144 units.


CHECK:

JO = OY

8x = 6x + 18

8(9) = 6(9) + 18

72 = 54 + 18

72 = 72

26. Mga salitang nagtatapos ng oy


Madaming salita ang nagtatapos sa “oy”.  Ilan sa mga ito ay apoy, kahoy,palaboy, baboy at kasoy. Ang mga salitang ito ay maaring tumutukoy sa ngalan ng tao, bagay, hayop, lugar at pangyayari. Sa talasalitaan ito ay tumutukoy sa pangngalan at bahagi ng pananalita. Kadalasan ng makikita sa unang bahagi ng pangungusap ang pangngalan pinaguusapan. Maari din gamitin ang mga salitang ito sa pang araw araw na gawaing bahay o pang araw araw na talasalitaan.  Ang mga pangungusap sa ibaba ay ginagamitan ng mga salitang nagtatapos sa “oy”.

Ang apoy ay may malaking bahaging ginampanan sa ating kasaysayan.Ang tuyong kahoy ay mainam na gawing haliging pintuan. Maraming palaboy sa bayan ang nagnanais na makaahon din sa kahirapan. Madaming alagang baboy ang naapektuhan ng epidemya. Ang kasoy ay isang uri ng tropikong prutas na kalinya ng pamilyang mani.

Mga uri ng pangngalan o “noun” sa wikang Ingles:

• Tao

• Bagay

• Pook

• Pangayayari
